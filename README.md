# acore_sod
LUA script for Azerothcore with ElunaLUA that gives buff which is Discoverer's Delight similar to SOD which is also base on player's level range for different Exp Bonus.

## Spell ID's
- 80865 = 200% / 1 - 49
- 80866 = 150% / 50 - 59
- 80867 = 100% / 60 = 69
- 80868 = 50% / 70 - 80
